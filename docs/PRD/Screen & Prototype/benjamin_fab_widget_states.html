Technické specifikace

Rozměry a barvy

  • • Standardní velikost: 56×56px
  • • Minimalizovaná velikost: 40×40px
  • • Primární barva: #5CB85C
  • • Hover barva: #45A049
  • • Chybová barva: #F44336
  • • Odznak: 16×16px červený (#F44336)

Animace a stavy

  • • Hover: Scale 1.05, silnější stín
  • • Active: Scale 0.95, tmavší barva
  • • Loading: Rotace + pulse efekt
  • • Přechod: 0.2s ease
  • • Stín: 0 2px 8px rgba(0,0,0,0.08)
  • • Font: System sans-serif

Pokyny k použití

Kdy použít výchozí stav:

Pro hlavní akce v aplikaci, dostupné kdykoliv

Kdy použít hover/active:

Vizuální zpětná vazba při interakci uživatele

Kdy použít loading:

Během zpracování požadavků nebo načítání dat

Kdy použít error:

Pro signalizaci chyb nebo kritických stavů
