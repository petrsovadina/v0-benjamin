Dokumentace komponent

Použití

Patient Context Banner je komponenta určená k zobrazení kontextových informací o pacientovi v lékařských aplikacích. Komponenta má tři hlavní stavy: úspěšné načtení dat, načítání a chybový stav. Každý stav má specifické vizuální indikátory a je optimalizován pro rychlou identifikaci stavu uživatelem.

Vlastnosti

  • • Šířka: Roztahuje se na plnou šířku kontejneru
  • • Pozadí: Světle šedé (#F5F5F5) pro kontrastní oddělení
  • • Layout: Ikona vlevo, informace uprostřed, status vpravo
  • • Barvy: Zelená pro úspěch, žlutá pro načítání, červená pro chybu
  • • Typografie: Systémové písmo, 14px pro hlavní text, 13px pro sekundární

Stavy komponent

Success State

Zobrazuje kompletní informace o pacientovi s zeleným indikátorem úspěchu.

Loading State

Animovaný spinner a placeholder text během načítání dat.

Error State

Chybová hlášení s možností opakování akce pomocí tlačítka.
