# Fix README.md Git Merge Conflict

## Overview
Hlavní soubor README.md obsahuje nevyřešené konflikty git merge (značky <<<<<<< HEAD na řádcích 3 a 168). To způsobuje, že vstupní bod projektu je pro nové vývojáře matoucí a brání správnému vykreslení dokumentace na GitHubu.


## Rationale

The README is the primary onboarding point for all developers. Having merge conflicts visible breaks trust, looks unprofessional, and makes it unclear which version of setup instructions to follow. This is a critical blocker for developer experience.
